This is an earthquake alert system that gets you IP and calls 'https://ipinfo.io/json/' for fetching your coordinates and then calls 'https://earthquake.usgs.gov/' with your coordinates and search radius of 1750km as query and takes action accordingly.

Libraries include : sys, time, json, requests, pygame, math, tkinter

To run :

Step 0) Ensure all above mentioned lib are installed and all the above project files are in the same folder.

Step 1) Open terminal in the directory where all the SeisGuard files are stored.

Step 2) Run the following command : python3 ./SeisGuard.py.
